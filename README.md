## 使い方
ファイルから入力
第一引数にファイルパスを渡します
```bash
& path_to_thisapp file
```
標準入力から入力
リダイレクトを使って標準入力から入力できます
```bash
& echo program >> path_to_thisapp
& path_to_thisapp < file
```
